School Performance Analytics Pipeline

PythonGoogle Sheets APIBigQueryPower BIStar schema

Problem

Academic performance data for 120+ schools across the district lived in scattered Google Sheets, updated inconsistently, with no single source of truth the District Collector or CEO could trust for decisions.

Architecture

  • Google Sheets API pulls raw entries from every school's register on a schedule.
  • Python extraction layer validates and hashes each row to detect real changes only.
  • BigQuery star schema with MERGE-based upsert logic keeps the warehouse current without full reloads.
  • Retry/backoff handling for API and load failures, so the pipeline recovers without manual intervention.
  • Power BI dashboard surfaces performance trends for district leadership.

Impact

Adopted by the District Collector and CEO as a standing reference for school performance across Kancheepuram — the first time this data has existed in one governed, queryable place.

ID02
Live · Production

Road Accident Intelligence Pipeline

PythonGeocodingHotspot clusteringPower BI

Problem

Road accident records existed as raw case entries with no spatial or trend view — no way to see where accidents cluster, or which stretches carry the highest fatality risk.

Architecture

  • Geocoded 7,500+ accident records to map every case to a precise location.
  • Built hotspot clustering to surface high-frequency accident zones across the district.
  • Layered in fatality-rate analysis to separate high-volume spots from high-severity ones.
  • Power BI dashboard delivered to district administration for road safety planning.

Impact

Dashboards were adopted directly by the district administration and are used in road safety planning decisions — turning raw case files into a spatial intelligence tool.
